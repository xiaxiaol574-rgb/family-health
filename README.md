# 家庭健康守护 | Health Guardian

> 💚 一个融合健康仪表盘 + 习惯追踪 + 急救知识库的家庭健康管理工具

## 功能

- 🏠 **健康仪表盘** — 健康评分、BMI、饮水追踪、体检倒计时、活动图表
- 📊 **习惯追踪** — 日历视图、每日记录（睡眠/运动/饮水/心情）、月度统计
- 📚 **健康知识库** — 急救指南、家庭药箱管理、季节健康贴士
- 👨‍👩‍👧‍👦 **多成员管理** — 支持多个家庭成员独立追踪

## 使用方法

直接在浏览器中打开 `index.html` 即可使用，无需安装任何依赖。

数据保存在浏览器本地（localStorage），无需网络连接。

## 技术栈

纯 HTML + CSS + JavaScript，零依赖。
